# redux-saga-login
A practice of redux-saga login flow

## Run
1. clone repo
    ```
    $ git clone https://github.com/SeanLin05583/redux-saga-login.git sean-redux-saga-login
    ```
2. cd sean-redux-saga-login
    ```
    $ cd sean-redux-saga-login
    ```
3. install package
    ```
    $ npm install
    ```
4. start project
    ```
    $ npm start
    ```
